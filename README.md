# cqf_implementation

Implementation of a cqf

## Compilation of the examples

From the project root
```bash
cmake -S . -B build
cmake --build build 
```
  
or   
  
From the project root
```bash
cmake -S . -B build
cd build && make 
```

## Unitary tests

From build directory
```bash
ctest
```


Binaries are in build/bin/
